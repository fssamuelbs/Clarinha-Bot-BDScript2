# 🤖 Clarinha Bot - BDScript2

Sistema de comandos desenvolvido em **BDScript2** para o bot **Clarinha** no Discord.

O projeto reúne diversos comandos interativos, jogos e sistemas de economia para servidores do Discord, utilizando a linguagem **BDScript2** através da plataforma **Bot Designer for Discord (BDFD)**.

---

## 📌 Sobre o Projeto

O **Clarinha Bot - BDScript2** é um repositório que organiza os scripts utilizados no bot Clarinha, incluindo:

* Comandos de interação entre usuários;
* Sistema de economia com moedas (Candy);
* Jogos interativos;
* Sistema de perfil;
* Sistema de loja e fazenda;
* Comandos utilitários para servidores.

O objetivo é **centralizar, organizar e versionar todos os comandos do bot** para facilitar manutenção, melhorias e colaboração.
